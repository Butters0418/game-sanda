# 週末 H5 聖誕小遊戲

針對嵌入 APP WebView 需求，以 Phaser 開發的輕量 H5 遊戲

## 🎯 遊戲玩法

1. **收集禮物**：左右控制正在跳躍的聖誕老人來收集禮物，同時閃躲炸彈
2. **獲勝條件**：遊戲時間 30 秒內，得分超過 300 分，即可獲得好禮
3. **失敗條件**：不小心收集到 3 個炸彈
4. **補充說明**：可鍵盤左右鍵操控

**🌐 線上展示**: [https://game-sanda.butters.idv.tw/](https://game-sanda.butters.idv.tw/)
